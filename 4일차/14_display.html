강강조 작은영역 
        ------------------
        display : 화면에 보이는 성격 (원래가지고 있는 성격을 화면에서만 바뀌어 보이도록 설정 (원래성격이 바뀌지 않음))
        - inline : 블럭성격을 인라인으로 변경, 거의 사용하지 않는다
        - block : 인라인을 블럭으로 변경**
                    화면에 보이기(js)
                    (js)none : 화면에서 사라지기
        - inline-block : 박스옆으로 나열. 10%사용
                        작은영역의 나열(상단유틸메뉴)
            float : 15% 
            주(flex) 80%
