1. hn: h1 ~ h6 : 제목
        2. p : 단락, 문단
        3. br : 강제개행(빈태그)
        4. ol : 순서 있는 목록
                자식으로 li 만 가능. li 자식으로 모든 태그 사용 가능
        5. ul : 순서 없는 목록 (비슷한 성격의 나열. 동일한 형태(성격)의 나열)
                자식으로 li 만 가능. li 자식으로 모든 태그 사용 가능
                - 예) 배너, 갤러리, 메뉴
        6. dl : 용어 정의 (자식 dt , dd 만 가능)
                dt 용어 - 안에는 인라인형식(strong, em 등)만 가능. 블록형태(hn, p 등)은 사용x
                dd 설명 - 자식으로 모든태그 가능
        7. img : 이미지
                속성 -  필수 : src (이미지주소) / alt (대체 텍스트)
                        선택 : width (너비) / height (높이) / title (도움말)
        8. a : 링크
                속성 -  href : url 주소 
                        target : _blank (새탭)
                        title : 도움말
        9. div : 공간분할 (의미없는 큰영역. 남용x)
                css화면결과 구조. 그룹핑
        10. strong : 강강조
        11. em : 약강조
        12. span : 의미없는 작은영역
        13. b : 의미없는 작은영역
        14. i : 폰트 아이콘
