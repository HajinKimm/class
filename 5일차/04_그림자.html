        box-shadow / text-shadow

        box-shadow
        가로거리 세로거리 블러 색상
        (px, %, 양수, 음수)
        가로거리 세로거리 블러 그림자크기 색상 (text-shadow(x))
            여러개를 ,로 나열하여 그림자를 여러개 만들 수 있음
        그림자코드 : https://getcssscan.com/css-box-shadow-examples






        /* inset은 box-shadow에서만 가능하다(text-shadow는 안됨) */
            /* box-shadow: inset 5px 5px 0 #dcdcdc; */
            /* box-shadow: inset 0 0 10px #333 */
            /* box-shadow: 0 0 0 10px #dcdcdc; */
            box-shadow: inset 0 0 10px 10px #dcdcdc;
