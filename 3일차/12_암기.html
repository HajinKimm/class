암기
        1. 블럭요소
            div, h1~h6, p, ul, ol, li, dl, dd
            1) 너비는 부모크기만큼 인식한다(100%) : 한줄
            2) 너비, 높이 가능
            3) padding, margin 가능

        2. 인라인요소
            a, span, em, b, i, strong
            1) 자신의 내용 크기만큼(컨텐츠 만큼) 인식 / 옆으로 나열
            2) 너비, 높이 사용불가
            3) margin, padding - 좌우만 가능
                단 : 상하가능하다 다른컨텐츠에 영향을 줄 수 없다

        3. 인라인 블럭요소
            img, dt, button, input
            1) 자신의 내용만큼만 인식(열으로 나열) - inline 성격
            2) 너비, 높이 가능 - block 성격
            3) margin / padding 가능 - block 성격
            4) 옆 사이 공백
                해결 : 부무의 요소에 font-size:0 처리
                       자신의 요소에 font-size 지정
